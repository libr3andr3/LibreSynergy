# libresynergy

Federated learning community suite. One command to deploy your own online school with SSO, courses, chat, live video, and crypto+fiat payments.

## Quick Start

```bash
pip install libresynergy
libresynergy bootstrap
```

## What's Included

- **Authentik** — Single Sign-On (SSO) with JWT tokens
- **Frappe LMS** — Course management and certificate issuance
- **Matrix + Element** — Federated chat rooms and spaces
- **Jitsi Meet** — Live video sessions and webinars
- **BTC PayServer** — Bitcoin + Monero payments
- **Stripe** — Credit card payments
- **Federation** — Opt-in P2P mesh between communities
